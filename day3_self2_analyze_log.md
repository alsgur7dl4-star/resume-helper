# Day 3 self2 분석 실행 메모

## 입력 문단과 키워드 선택 이유

입력 문단은 제가 실제로 경험한 SI/SM 인턴 업무와 현재 배우고 있는 MCP·AI 서비스 개발 방향을 함께 보여줄 수 있다고 생각해서 선택했습니다.

입력 키워드는 `Java, SI/SM, 배치, 오류 처리, 기능 통합, MCP, AI 서비스`로 정했습니다. Java와 SI/SM은 인턴 경험을 보여주기 위한 키워드이고, 배치·오류 처리·기능 통합은 실제 맡았던 업무를 설명하기 위한 키워드입니다. MCP와 AI 서비스는 앞으로 확장하려는 학습 방향을 보여주기 위해 넣었습니다.

## 실행 결과 해석

분석 결과 score는 71점으로 나왔고, 키워드 7개 중 5개가 매칭되었습니다. `Java`는 원문에서 `자바`로 표현되어 missing 처리되었고, `오류 처리`도 원문에서는 `오류를 처리하는`으로 표현되어 missing 처리된 것 같습니다.

현재 코드는 동의어 처리 없이 단순 부분일치 방식으로 동작하기 때문에 이 결과는 정상이라고 생각합니다.

탐지된 결함은 `STAR/PREP 프레임 미준수`, `NCS 키워드 누락`입니다. 현재 문단에는 경험은 들어 있지만 상황, 행동, 결과가 명확하게 나뉘어 있지는 않아서 STAR/PREP 프레임 미준수는 맞는 결과라고 봤습니다.
