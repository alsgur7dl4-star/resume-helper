# day2_self1_style_log.md

## 실행 환경

- OS: Windows
- Python: 3.11
- 실행 명령: `uv run python resume_helper.py`

## 스타일 선택 로그

- `/style` 입력 결과: 사용 가능한 스타일 목록이 출력되고 현재 스타일은 간결형으로 유지됨
- `/style 간결형` 입력 결과: 간결형 스타일로 변경됨
- `/style 스토리형` 입력 결과: 스토리형 스타일로 변경됨
- `/style 직무맞춤형` 입력 결과: 직무맞춤형 스타일로 변경됨
- `/style 없는스타일` 입력 결과: 알 수 없는 스타일이라는 안내와 사용 가능한 스타일 목록이 출력되고 기존 스타일이 유지됨
